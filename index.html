GPU industry research repo

Market research on GPU compute, model economics, and ICN positioning.

This repository is a browser-first research workspace. It combines presentation decks, strategy dashboards, supporting notes, and structured data so the story can be explored from multiple angles: what the hardware does, how the market is structured, and where the economics are moving.

Presentation hub in index.html Three main narrative tracks GitHub Pages friendly Built for internal walkthroughs

What this repo contains

3 Primary decks: foundations, competitive landscape, trends and positioning.
4+ Strategy dashboards for economics, value chain, taxonomy, and intelligence.
JSON Structured data files that power the interactive charts and comparisons.
Notes Human research logs, interview notes, and source material for the decks.

Recommended viewing order

Start with the hub, then follow the sequence below if you want the cleanest narrative from hardware basics to market strategy.
